это всё sign - знаки:
Less Than (меньше чем) - &lt; в HTML это выглядит так <
Less or Equal (меньше или равно) - &le; в HTML это выглядит так ≤
Greater Than (больше чем) - &gt; в HTML это выглядит так >
Greater or Equal (больше или равно) - &ge; в HTML это выглядит так ≥
Equal to (эквивалентно) - &Equal; в HTML это выглядит так ⩵;
Equals (эквивалентны - равны) - &equals; - именно с маленькой, в HTML это выглядит так =


Unordered List - <ul> - Ненумерованный список - маркированный список
Ordered List - <ol> - Нумерованный список
Элементы Ненумерованного и Нумерованного списка:
List Items - <li> - элемент списка

Description List - <dl> - Список определений
Элементы Списка определений:
Definition Term - <dt>
Definition Description - <dd> - contains data that describes a definition term
