Линейное Программирование



Данный сайт содержит информацию для студентов факультета прикладной математики.
Информация для выполнения лабораторных работ по решению задач линейного программирования. На примере задач минимизации потребления ресурсов на предприятии


  • Инструкции к выполнению лабораторной работы в pdf или docx
  • SimplexMethod_CMD - консольная версия приложения, её легче понять.
  • SimplexMethod_GUI - оконная версия программы. Файлы для тестов лежат в архиве "fileTest1.txt", "fileTest2.txt"



P.S. для запуска проектов нужна VS. Если VS не установлена и вы не хотите её устанавливать или имеете альтернативную IDE , то можно достать исходники из проекта и скомпилировать вручную.
