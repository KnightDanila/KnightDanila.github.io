HOW TO USE
Steps:
  1. Click on "Drag and Drop ZONE" and select something
  2. Or drop something on "Drag and Drop ZONE"
  3. Wait... After loading you will get what you need
  4. CHAR(0-255) - represent values in integer data 0-255
  5. HEX - represent values in hexadecimal
  6. Base64 - Base64 encoding data
  7. decode Base64 - raw (text) - decoding data from Base64 and show raw data
